# 07 - Lists and Collections

Lists store multiple values in order.

```python
names = ["Ali", "Mona", "Sam"]
print(names[0])

names.append("Nora")
names.remove("Sam")
```

Loop through a list:

```python
for name in names:
    print(name)
```

Useful operations:

```python
len(names)
"Ali" in names
```
