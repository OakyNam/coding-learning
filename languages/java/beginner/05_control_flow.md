# 05 - Control Flow

Use `if`, `else if`, and `else` for decisions.

```java
int score = 82;

if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("Keep practicing");
}
```

Use `.equals()` to compare strings:

```java
if (name.equals("Mona")) {
    System.out.println("Welcome");
}
```
