# 02 - Project and File Structure

A small Python project can start with one folder and one `.py` file.

```text
my-python-app/
  README.md
  main.py
```

`main.py` is a common place to start a beginner program.

```python
print("Starting app")
```

As projects grow, split related code into more files. Keep names lowercase and descriptive, such as `calculator.py`.

Practice: create a folder with `README.md` and `main.py`, then run `main.py`.
