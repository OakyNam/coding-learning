# 02 - Project and File Structure

A tiny C project can start with one source file.

```text
hello-c/
  README.md
  main.c
```

As projects grow, split declarations into `.h` header files and implementations into `.c` source files.

```text
calculator/
  main.c
  calculator.c
  calculator.h
```

Practice: create a folder with `main.c` and compile it from the terminal.
