# 06 - Loops

Use `for` when you know how many times to repeat.

```csharp
for (int i = 1; i <= 3; i++)
{
    Console.WriteLine(i);
}
```

Use `while` while a condition remains true.

```csharp
int count = 0;
while (count < 3)
{
    Console.WriteLine(count);
    count++;
}
```
