# 04 - Input and Output

Use `printf` for output.

```c
printf("Hello\n");
```

Use format specifiers for values:

```c
int age = 25;
printf("Age: %d\n", age);
```

Use `scanf` for simple input:

```c
int number;
scanf("%d", &number);
printf("You entered %d\n", number);
```
