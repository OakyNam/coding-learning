# 05 - Control Flow

Use `if`, `else if`, and `else` for decisions.

```javascript
const score = 82;

if (score >= 90) {
  console.log("A");
} else if (score >= 80) {
  console.log("B");
} else {
  console.log("Keep practicing");
}
```

Use `===` for equality checks.

Practice: check whether a number is even or odd.
