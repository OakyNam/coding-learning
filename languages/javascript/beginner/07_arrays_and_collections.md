# 07 - Arrays and Collections

Arrays store ordered values.

```javascript
const names = ["Ali", "Mona", "Sam"];
console.log(names[0]);

names.push("Nora");
```

Useful methods:

```javascript
names.includes("Ali");
names.length;
names.map(name => name.toUpperCase());
```
