# 09 - Functions

Functions name reusable blocks of code.

```python
def greet(name):
    return f"Hello, {name}!"

message = greet("Ali")
print(message)
```

Functions can take parameters and return results.

```python
def add(a, b):
    return a + b
```

Good functions usually do one clear job.
