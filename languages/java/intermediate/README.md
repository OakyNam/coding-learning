# Java Intermediate

This level builds practical Java applications with objects, errors, files, and tests.

## Planned Lessons

1. Classes and constructors
2. Encapsulation
3. Inheritance and interfaces
4. Collections framework
5. Exceptions
6. File I/O
7. Generics
8. Unit testing
9. Maven or Gradle
10. Building HTTP APIs

## Goal

You should be able to build a small Java project with clear classes and tests.
