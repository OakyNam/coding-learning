# C#

C# is part of the Coding Learning language curriculum. Use the beginner path to learn syntax, the intermediate path to build useful programs, and the advanced path to reason about larger systems.

## Levels

- [Beginner](beginner/README.md)
- [Intermediate](intermediate/README.md)
- [Advanced](advanced/README.md)

## Suggested Path

1. Finish beginner lessons in order.
2. Build two small programs without copying examples.
3. Move into intermediate lessons and connect this language to shared topics like JSON, Git, testing, and APIs.
