# 02 - Project and File Structure

A tiny Java program can start with one file.

```text
hello-java/
  Hello.java
```

Class names usually use PascalCase and match the file name.

Larger projects often use this shape:

```text
src/
  main/
    java/
      app/
        Main.java
```

Practice: create `Main.java` with a `main` method and run it.
