# 03 - Variables and Data Types

Java variables have explicit types.

```java
String name = "Mona";
int age = 25;
double price = 19.99;
boolean active = true;
```

Common beginner types:

- `String`
- `int`
- `double`
- `boolean`
- `char`

Practice: create variables for a course name, lesson count, and completion status.
