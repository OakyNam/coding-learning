# 03 - Variables and Data Types

Variables store values so you can reuse them.

```python
name = "Mona"
age = 25
price = 19.99
is_active = True
```

Common beginner types:

- `str` for text
- `int` for whole numbers
- `float` for decimals
- `bool` for true or false

Use `type()` to inspect a value:

```python
print(type(name))
```
