# Python Intermediate

This level builds from scripts into maintainable programs.

## Planned Lessons

1. Virtual environments and dependency files
2. Reading and writing files
3. Exceptions and error handling
4. Classes and objects
5. Type hints
6. Working with JSON and APIs
7. Testing with `pytest`
8. Project organization
9. Useful standard library modules
10. Packaging small tools

## Goal

You should be able to build a small command-line app with tests and clear structure.
