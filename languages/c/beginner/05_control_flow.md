# 05 - Control Flow

Use `if`, `else if`, and `else` for decisions.

```c
int score = 82;

if (score >= 90) {
    printf("A\n");
} else if (score >= 80) {
    printf("B\n");
} else {
    printf("Keep practicing\n");
}
```

Use `==`, `!=`, `>`, `<`, `>=`, and `<=` for comparisons.

Practice: check whether a number is positive, negative, or zero.
