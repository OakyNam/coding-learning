# 03 - Variables and Data Types

C variables need explicit types.

```c
int age = 25;
double price = 19.99;
char grade = 'A';
```

Strings are arrays of characters:

```c
char name[] = "Mona";
```

Common types:

- `int` for whole numbers
- `float` and `double` for decimals
- `char` for one character

Practice: create variables for a product count, price, and grade.
