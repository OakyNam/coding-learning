# 10 - Packages and Imports

Packages organize Java classes.

```java
package app.tools;

public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
}
```

Imports bring classes into a file.

```java
import java.util.ArrayList;
import java.util.Scanner;
```

Build tools like Maven and Gradle help manage dependencies.

Practice: import `ArrayList` and create a list of three names.
