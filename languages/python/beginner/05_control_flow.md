# 05 - Control Flow

Control flow lets a program make decisions.

```python
score = 82

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("Keep practicing")
```

Use comparison operators such as `==`, `!=`, `>`, `<`, `>=`, and `<=`.

Use logical operators:

```python
if score >= 80 and score < 90:
    print("Solid work")
```
