# 04 - Input and Output

Use `console.log` for output.

```javascript
console.log("Hello");
```

In browsers, input often comes from forms or prompts.

```javascript
const name = prompt("Name:");
console.log(`Hello, ${name}!`);
```

In Node.js beginner examples, start with fixed values before learning input libraries.

Practice: print a sentence using a variable inside a template string.
