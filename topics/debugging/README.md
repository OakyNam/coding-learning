# Debugging

Debugging is the process of finding why a program behaves differently from what you expected.

## Core Ideas

- Reproduce the problem.
- Read the error message carefully.
- Isolate the smallest failing example.
- Change one thing at a time.

## Useful Techniques

- Print variable values.
- Use a debugger and breakpoints.
- Add tests for the failing case.

## Practice

Take a broken loop and print each variable during every iteration.
