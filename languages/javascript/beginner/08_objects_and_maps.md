# 08 - Objects and Maps

Objects store named values.

```javascript
const student = {
  name: "Mona",
  age: 25,
  active: true
};

console.log(student.name);
```

Update values:

```javascript
student.city = "Chicago";
student.age = 26;
```

Practice: create an object for a book with title, author, and page count.
