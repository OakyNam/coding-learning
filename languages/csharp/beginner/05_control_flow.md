# 05 - Control Flow

Use `if`, `else if`, and `else` to choose what happens.

```csharp
int score = 82;

if (score >= 90)
{
    Console.WriteLine("A");
}
else if (score >= 80)
{
    Console.WriteLine("B");
}
else
{
    Console.WriteLine("Keep practicing");
}
```

Practice: check whether a user is old enough to vote.
