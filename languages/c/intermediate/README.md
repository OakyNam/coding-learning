# C Intermediate

This level builds stronger C programs with memory, files, and multi-file structure.

## Planned Lessons

1. Pointers
2. Dynamic memory with `malloc` and `free`
3. Strings in depth
4. Structs and pointers
5. Header/source organization
6. File I/O
7. Error handling
8. Build tools and Makefiles
9. Debugging with a debugger
10. Testing C programs

## Goal

You should be able to build and debug a small multi-file C project.
