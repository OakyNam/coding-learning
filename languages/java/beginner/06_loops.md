# 06 - Loops

Use `for` when you know how many times to repeat.

```java
for (int i = 1; i <= 3; i++) {
    System.out.println(i);
}
```

Use `while` while a condition is true.

```java
int count = 0;
while (count < 3) {
    System.out.println(count);
    count++;
}
```
