# 06 - Loops

Use `for` to repeat a known number of times.

```javascript
for (let i = 1; i <= 3; i++) {
  console.log(i);
}
```

Loop through arrays:

```javascript
const names = ["Ali", "Mona", "Sam"];

for (const name of names) {
  console.log(name);
}
```
